## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Set up environment variables:**
    ```bash
    cp .env.example .env
    ```
    Fill in your environment variables in the `.env` file.
3.  **Set up the database:**
    ```bash
    npm run db:push
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
